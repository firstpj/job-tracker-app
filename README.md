# Job Application Tracker

A simple web application built with React and TypeScript to track job applications.

## Features

* Add job applications
* Edit job details
* Delete jobs
* Persistent storage using localStorage

## Tech Stack

* React
* TypeScript
* Vite

## How to Run

1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open in browser:
   http://localhost:5173/
